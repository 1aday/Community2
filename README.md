# Community2

Person search and profile tool with AI-generated profile pictures and history tracking.

## What it does

Community2 is a person search and profile management application that allows users to look up individuals and view detailed profiles. It features AI-generated profile pictures, search history tracking, and a clean interface built with shadcn/ui components. The app provides a streamlined workflow for discovering and organizing information about people.

## Tech Stack

- **Next.js** — React framework for the web application
- **TypeScript** — Type-safe development
- **shadcn/ui** — Accessible, customizable UI components

## Getting Started

```bash
git clone https://github.com/1aday/Community2.git
cd Community2
npm install
npm run dev
```

## Environment Variables

None required for basic usage.
